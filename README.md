coers
=====

Coers is a very small library to provide small coercion
on primitive type in Erlang. This library was built
essentially for internal tools at derniercri.io

Build & test
-----
    $ # Compile the library
    $ rebar3 compile
    $ # run the tests using eUnit
    $ rebar3 eunit
